# AJStorybook
The Storybook of AllJoint
傲嬌前端範例集

## Getting Started
準備開始
```bash
git clone https://github.com/AllJointTW/AJStorybook.git
cd AJStorybook
npm install
npm run storybook
```

## Contribute
PLZ Follow the Github Flow!!
請遵守 Github Flow！！

## Style Format
自動格式化
```bash
npm run prettier
```

## Build Static
編譯靜態網頁，考慮做成 Github Page
```bash
npm run build-storybook
```

## Clean
清除編譯好的靜態網頁
```bash
npm run clean
```
